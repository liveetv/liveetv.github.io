Covid19 Data API

You can get Covid19 data using this API. There are two version of this API. Maybe in future I will add more. Read below information carefully.
Endpoints:
  1. "/covid19/v1" - Version 1 Covid19 Data API
  2. "/covid19/v2" - Version 2 Covid19 Data API
v1 Args:
  1. "country" - Pass Country Name (Optional)
v2 Args:
  1. "country" - Pass Country Name (Required)
  2. "from" - "2021-07-21T00:00:00Z" (Optional)
  3. "to" - "2021-07-24T00:00:00Z" (Optional)
Method - GET
